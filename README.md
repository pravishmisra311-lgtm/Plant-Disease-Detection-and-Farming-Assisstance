# 🌿 Plant Disease Detection and Farming Assistance

A responsive web application that helps users identify plant diseases using AI-powered image classification and provides farming assistance through crop-specific tips and government schemes.

## 🔗 Live Demo

https://agrovision-ai-psi.vercel.app/#/

---

## 📌 Features

- 🌱 Upload plant leaf images for disease detection
- 🤖 AI-powered disease prediction using TensorFlow.js
- 📋 Displays predicted disease with confidence score
- 🌾 Crop-specific farming recommendations
- 💡 Useful farming tips for better crop management
- 🏛️ Government schemes information for farmers
- 🌙 Dark mode support
- 📱 Responsive design for desktop and mobile devices

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/pravishmisra311-lgtm/Plant-Disease-Detection-and-Farming-Assistance.git
```

Go to the project folder

```bash
cd Plant-Disease-Detection-and-Farming-Assistance
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

The application will start on:

```
http://localhost:5173
```
